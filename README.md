# 有害生物数据库处理系统




目前版本为 `v4.0+` 基于 `vue-cli` 进行构建



## 启动

```bash

# 进入项目目录
cd vue-weed

# 安装依赖
npm install


# 启动服务
npm run dev
```

浏览器访问 [http://localhost:9528](http://localhost:9528)

## 发布

```bash
# 构建测试环境
npm run build:stage

# 构建生产环境
npm run build:prod
```

## 其它

```bash
# 预览发布环境效果
npm run preview

# 预览发布环境效果 + 静态资源分析
npm run preview -- --report



# 代码格式检查并自动修复
npm run lint -- --fix
```
